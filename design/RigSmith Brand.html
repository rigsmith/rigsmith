01 The mark
[ ] node harness

A harness,
not a hull.

Every RigSmith tool lives inside a pair of brackets — the rig that holds your setup together. It reads as code the instant you see it, and it's the one element that never changes across the family.

At the center sits the node: the core rig. Each tool swaps that node for its own glyph, so the set stays unmistakably related while every member keeps its own identity.

02 Construction & clearspace
Anatomy
Two brackets on a 100-unit grid, 8-unit stroke, rounded joins. The glyph occupies the central 30 units.
Clearspace
Keep clearspace equal to one bracket arm on every side. Nothing crosses the boundary.
Min size
The mark holds down to 16px. Below that, use the solid tile so the glyph stays legible.
03 Color

One accent per tool, all built on a shared oklch(0.70 0.15 h) — identical lightness and chroma, hue is the only variable. They sit together without one shouting over another.
