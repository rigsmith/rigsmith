Getting started / Install

Install

Every RigSmith tool is a single, statically-linked Go binary with zero runtime dependencies. Install the whole family with one command, or grab a tool on its own.

#Quick install

The install script detects your OS and architecture, then drops the four binaries on your PATH.

terminalcopy
$ curl -fsSL https://rigsmith.dev/install.sh | sh
  resolving latest…        v1.4.0
  ✓ rig        → /usr/local/bin/rig
  ✓ changeRig  → /usr/local/bin/changeRig
  ✓ shipRig    → /usr/local/bin/shipRig
  ✓ claudeRig  → /usr/local/bin/claudeRig
Tip

Run rig doctor right after install to confirm your toolchain is wired up correctly.

#Package managers

Prefer a package manager? The family is published to Homebrew and Scoop.

macos · linux · windows
# macOS / Linux
$ brew install rigsmith

# Windows
$ scoop install rigsmith

#Verify

CommandWhat it shows
rig --versionInstalled version of the launcher
rig doctorEnvironment & toolchain health check
rig --helpAll verbs for the detected stack
Note

On first run inside a repo, rig caches the detected stack in .rig/. Add it to your .gitignore.
